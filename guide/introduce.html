# 介绍

medaxis-web-hpams (opens new window)是三级医院管控系统_新华版，结合Vue技术栈、axios、ES2015+和element-ui组件库实现。

# 目录结构

此项目开发框架完善，只是经过多人处理之后，某些文件结构冗余，后期会逐步优化，下面是整个项目的目录结构。

├── public                            # 静态资源
├── src                               # 工程主目录
│   ├── api                           # api请求
|   |   ├── *                         # 其他api模块
│   │   └── index.js                  # api入口文件
│   ├── assets                        # 存放各种静态资源
│   ├── components                    # 公共组件
│   │   ├── base                      # 基础组件
│   │   │   ├── base-name             # 组件名称目录
│   │   │   │   ├── *                 # 组件私有文件或子组件
│   │   │   │   └── index.vue         # 组件入口文件
│   │   │   └── index.js              # 基础组件入口文件 （以插件形式导出，在全局注册）
│   │   └── business                  # 业务组件
│   │       └── name                  # 组件名称目录
│   │           ├── *                 # 组件私有文件或子组件
│   │           └── index.vue         # 组件入口文件
│   ├── config                        # 冗余文件（待优化）
│   ├── directive                     # 指令
│   ├── enum                          # 枚举类型
│   ├── filter                        # 过滤器
│   ├── mixins                        # 混入
│   ├── mock                          # 实际上存放的是json文件，并未接入mock
│   ├── router                        # 路由
│   ├── store                         # 状态管理
│   ├── utils                         # 工具
│   ├── views                         # vue文件
│   ├── App.vue                       # app根节点
│   ├── main.js                       # 入口文件
│   └── permission.js                 # 冗余文件（待优化）
├── .editorconfig                     # 插件 EditorConfig for VS Code 配置文件
├── .env                              # 默认环境变量
├── .env*                             # 其他环境变量
├── *                                 # 其配置文件
├── .eslintrc.js                      # eslint规则配置
├── .babel.config.js                  # babel配置文件
├── .package.json                     # npm配置文件
└── .vue.config.js                    # vue配置文件（包含webpack）
